## 📖 Project Overview

The **Zynxis Client Dashboard** is a professional business management platform designed to provide a centralized workspace for monitoring key performance indicators, managing clients, tracking projects, organizing tasks, and visualizing business data.

The application follows modern frontend development practices with a strong focus on:

- Clean UI/UX
- Responsive Design
- Performance Optimization
- Accessibility
- Component Reusability
- Professional Dashboard Experience

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/rashid1351/zynxis-internship-muhammad-rashid-azam.git
```

### Navigate to Project

```bash
cd Week8_Final_Dashboard_Project
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

---

## 🏗️ Production Build

Build the project:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
