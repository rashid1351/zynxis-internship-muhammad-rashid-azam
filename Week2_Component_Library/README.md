# Zynxis Frontend Internship Task – Week 2

## Component-Based Design System

A modern React.js / Next.js component library built using a slate-grey dark theme inspired by the Zynxis brand. This project demonstrates reusable UI architecture, component-driven development, responsive design, and modern frontend best practices.

---

## Project Overview

This project was developed as part of the Zynxis Frontend Development Internship Program (Week 2 – Component-Based Design).

The objective was to build a reusable UI component library using React.js or Next.js and create a consistent slate-grey dark-themed design system.

## Installation

Clone the repository:

```bash
git clone <https://github.com/rashid1351/zynxis-internship-muhammad-rashid-azam/tree/main/Week2_Component_Library>
```

Navigate to project directory:

```bash
cd project-name
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
